# dotnet-clean-architecture-boilerplate
A boilerplate with standard projects that implement the clean architecture pattern. Feel free to use this as a starting point for all your projects

Copy the link below to clone this template
```
https://github.com/jamattey/dotnet-clean-architecture-boilerplate.git
```

This template was developed with Visual Studio Code, hence there might be a few hitches using it with Visual Studio, but it should work.
